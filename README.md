# Hospital Intelligence MY

A personal interactive clinical reference tool mapping public and private tertiary hospitals across Malaysia. Covers bed counts, specialty rankings, technology and robotics, insurance compatibility, and insider navigation hacks. All data verified against MOH Malaysia, Clinical Research Malaysia, and official hospital sources (2024–2025).

## Usage

Open `index.html` directly in any modern browser — no server required.

To deploy via **GitHub Pages**: go to repo Settings → Pages → Source: `main` branch → root (`/`). The tool will be live at `https://[your-username].github.io/hospital-intelligence-my/`.

## Stack

Single-file static HTML — no build step required.

- [Tailwind CSS](https://tailwindcss.com/) (CDN)
- [Lucide Icons](https://lucide.dev/) (CDN)
- [DM Sans / DM Mono](https://fonts.google.com/) via Google Fonts
- Vanilla JS (inline)

## Disclaimer

For personal reference only. Verify all clinical and insurance details directly with the respective hospital before making any medical decisions.
