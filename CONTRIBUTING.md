# Contributing to Hospital Intelligence Malaysia

> Healthcare information should be easier to find, verify, understand, and compare.

Hospital Intelligence Malaysia started as a simple idea. We're building a community-driven platform that improves access to reliable hospital and healthcare information for Malaysians — and we welcome contributors across many disciplines.

## How to Contribute

### Data Contributions
Hospital records, pricing data, and service information must be verified against official sources before submission. See [`src/data/`](./src/data/) for the data structure.

### Code Contributions
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes and verify the build: `npm run build`
4. Open a pull request with a clear description of what changed and why

### Reporting Issues
Found incorrect hospital information, a broken link, or outdated data? Open an issue with:
- The specific hospital or data point
- What is incorrect
- Your source for the correct information

---

## Data Quality Standards

This platform serves real patients making real healthcare decisions. All data must be:
- Verified against MOH Malaysia official records, hospital websites, or other authoritative sources
- Cited with a source link where possible
- Flagged clearly if estimated or unverified

**Do not submit unverified data.** When in doubt, leave a note in your PR and we'll verify together.

## Vision

We believe healthcare information should be:
- **Transparent** — costs, quality, and capability should be visible to patients
- **Accessible** — in plain language, across languages where possible
- **Useful** — actionable, not just informational
- **Trusted** — rigorously verified, clearly sourced

This is an open community initiative. The vision, roadmap, and priorities are shaped together.

If you believe healthcare information should be more transparent, accessible, and useful for everyone — **we'd love to connect.**
